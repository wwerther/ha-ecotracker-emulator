# Documentation

Supplementary docs for the EcoTracker Emulator integration. Anything that is useful for
developers or curious users but does not belong into the user-facing `README.md` /
`info.md` lives here.

## Contents

- [`api-spec.md`](api-spec.md) – Original EcoTracker `/v1/json` API specification (the
  device behaviour we emulate). **Source of truth** for what fields exist and what they
  mean.

## Conventions

- Keep documents in English by default; manufacturer quotes / verbatim specs may stay in
  their original language and should be marked as such at the top of the file.
- When adding new docs, link them from this index.
